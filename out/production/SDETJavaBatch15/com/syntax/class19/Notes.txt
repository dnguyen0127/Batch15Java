Constructor - block of code similiar to a method that is executed when we create an object


Why do we need it? - constructor initializes the object

How to create a constructor?

1. Constructor name MUST be same as a class
2. No return type

Type of constructors:
non argument
parametrized


this keyword - is used to refer to the current object


this can be used with variables and methods

this. used to refer to the instance variables
In situations when local variable have same name as instance ->
we use this. to differentiate and refer to the instance variable

this can be used to refer to the instance methods as well
